# 🩸 Hemoglobin Disorder Classification

This project applies Machine Learning and Neural Network models to classify different types of hemoglobin disorders using medical and laboratory data.

## 📊 Dataset
- 3000 rows
- 34 columns
- Target variable: `variant_type`

The dataset contains blood measurements, clinical indicators, and hemoglobin-related features.

## 🤖 Models Used
- Logistic Regression
- Random Forest
- K-Nearest Neighbors (KNN)
- Neural Network (TensorFlow/Keras)

## 📈 Evaluation Metrics
The models were evaluated using:
- Accuracy
- Precision
- Recall
- F1-score

## 🏆 Best Model
Random Forest achieved the best overall performance on this dataset.
